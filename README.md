# Лабораторная Bash

---

## Цель: 
### Реализовать основные команды управления файлами и каталогами bash

---
### Поддерживаемые команды:
1. ```ls``` - список файлов и катологов (с поддержкой ```-l```)
2. ```cd``` - смена катологов (с поддержкой ```~``` ```..```)
3. ```cat``` - вывод содержимого
4. ```cp``` - копирование файлов и каталогов (с поддержкой ```-r``` для рекурсивного копирования)
5. ```mv``` - перемещение и переименование файлов
6. ```rm``` -  удаление файлов и каталогов (с флагом ```-r``` и подтверждением для каталогов)
7. ```zip``` - архивирование в .zip-файл
8. ```unzip``` - разархивирование из .zip-файла
9. ```tar``` - архивирование в .tar.gz-файла
10. ```untar``` - разархивирование из tar.gz-файла
11. ```grep``` - поиск строк, соответствующих паттерну (с поддержкой ```-r``` ```-i```)
---

### Запуск:
```
python -m src.main.py
```
